Advanced Auto-Scan System

This console is configured to **automatically sync** tracks from your local folders on page load:

1. Server Auto-Scanning: If hosted locally via Live Server or on any web server, AURA automatically scans the Track/ folder using background HTML index parser. Any MP3s or WAV files added there will load natively without changes!

2. Predefined Config: To support running directly via file:// protocol, you can list your file names inside the CONFIG_TRACKS array at the top of the Javascript code.

3. Manual Sync Fallback: If security constraints prevent automated scanning, click "Manual Sync", point to your local Track folder, and it will sync immediately.
